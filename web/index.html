Rugburn - Patch for PangYa™

Rugburn is a patch for PangYa™ which does the following:

  • Disable GameGuard: Rugburn completely disables GameGuard and prevents it from starting up.
  • Redirect Network Requests: Rugburn can redirect network requests to addresses and URLs, with configurable mappings, making it easier to set up clients for private servers.
  • Bypass Updater: Rugburn will allow you to run the game file (ProjectG.exe) directly, so you don't need to worry about the updater anymore.
In short, Rugburn is an all-in-one patch that makes it possible to easily set up stock, original PangYa clients to run with custom servers. Rugburn is free, open source software. You may share it as you wish under the license terms.

How It Works

Rugburn is a patch for ijl15.dll, an innocuous JPEG decoding library included with PangYa. When the PangYa client loads a Rugburn-patched ijl15.dll, Rugburn will patch various Windows API functions to carefully disable GameGuard and redirect network requests. In addition, Rugburn will apply a patch to disable the GameGuard health checking routines, so that the game will continue running without GameGuard.

Please note: Although Rugburn disables GameGuard, it is not cheating software and cannot be used to facilitate cheating. It fully disables GameGuard and thus can not be used on any official servers.

Patch

This tool will help you patch your PangYa™ installation with Rugburn. Please note that this tool works entirely offline in your browser and does not require the internet after it is loaded. No data is uploaded to the Internet when using this tool.

If you are a more advanced user, you can also download the source code and build Rugburn manually.
